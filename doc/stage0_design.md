# 阶段 0：基础 FP32 FMA 设计文档

## 1. 模块概述

本阶段实现基准 FP32 FMA（Fused Multiply-Add）运算单元，完成公式：

$$result_{[31:0]} = A_{[31:0]} \times B_{[31:0]} + C_{[31:0]}$$

采用三级流水线架构：Stage1 = 乘法，Stage2 = 对阶/CSA，Stage3 = 规格化/舍入。

## 2. 模块接口

### 输入端口

| 端口名 | 位宽 | 描述 |
|--------|------|------|
| `clk` | 1 | 系统时钟 |
| `rst_n` | 1 | 异步低电平复位 |
| `valid_in` | 1 | 输入数据有效信号 |
| `A` | [31:0] | FP32 乘数 A |
| `B` | [31:0] | FP32 乘数 B |
| `C` | [31:0] | FP32 加数 C |

### 输出端口

| 端口名 | 位宽 | 描述 |
|--------|------|------|
| `result` | [31:0] | FP32 运算结果 |
| `valid_out` | 1 | 输出数据有效信号 |
| `overflow` | 1 | 上溢标志 |
| `underflow` | 1 | 下溢标志 |
| `inexact` | 1 | 非精确结果标志 |

## 3. FP32 位域定义

| 字段 | 位段 | 位宽 | 说明 |
|------|------|------|------|
| 符号位 | `[31]` | 1 bit | 0 = 正，1 = 负 |
| 指数位 | `[30:23]` | 8 bit | bias = 127 |
| 尾数位 | `[22:0]` | 23 bit | 含隐含 1（规格化数） |

## 4. 三级流水线数据通路

### Stage 1：乘法阶段

- **符号位异或**：`sign_product = A[31] ^ B[31]`
- **指数相加**：`exp_sum[8:0] = A[30:23] + B[30:23] - 8'd127`（bias 修正，9bit 防溢出）
- **尾数乘法**：`{1'b1, A[22:0]} × {1'b1, B[22:0]}` → `product[47:0]`（48 bit 乘积）

### Stage 2：对阶/CSA 阶段

- **对阶偏移量**：`shift_amt = exp_sum - C_exp`（C 的指数）
- **移位器对齐**：将 C 尾数按偏移量右移对齐到乘积
- **CSA 压缩**：3-2 CSA 将乘积高位与对齐后的 C 尾数压缩为 `sum[49:0]` 和 `carry[49:0]`

### Stage 3：规格化/舍入阶段

- **加法**：`sum + carry` → 原始结果
- **前导零检测（LZD）**：检测结果最高有效位位置
- **规格化**：左移消除前导零，调整指数
- **舍入**：Round to Nearest Even（RNE）
- **输出**：`result[31:0]` = `{sign, exp[7:0], mantissa[22:0]}`

## 5. 关键运算器清单

| 运算器 | 数量 | 位宽 | 用途 |
|--------|------|------|------|
| 乘法器 | ×1 | 48 bit | 尾数乘法 `24b × 24b → 48b` |
| 移位器 | ×1 | 50 bit | 对阶移位（C 尾数对齐） |
| 3-2 CSA | ×1 | 50 bit | 压缩乘积与 C 尾数 |
| 加法器 | ×1 | 50 bit | 最终累加 |
| 规格化/LZD | ×1 | — | 前导零检测与规格化 |

## 6. 特殊值处理

| 输入条件 | 输出结果 |
|----------|----------|
| A 或 B 为 NaN | NaN（`32'h7FC00000`） |
| C 为 NaN | NaN |
| A 或 B 为 Inf，另一个为零 | NaN（0 × ∞ 未定义） |
| A 或 B 为 Inf，C 为 Inf 且符号相反 | NaN（∞ - ∞ 未定义） |
| A 或 B 为 Inf | Inf（符号由乘积决定） |
| C 为 Inf | Inf（C 的符号） |
| A 或 B 为零 | C（乘积为零） |
| 结果指数 > 254 | overflow = 1，返回 Inf |
| 结果指数 < 1 | underflow = 1，返回零或 denormal |
| 舍入导致精度损失 | inexact = 1 |

## 7. 模块层次

```
fma_fp32_stage0
├── 输入解析（符号、指数、尾数提取）
├── Stage 1 流水线寄存器
│   ├── 48bit 乘法器
│   └── 指数加法器
├── Stage 2 流水线寄存器
│   ├── 50bit 移位器
│   └── 3-2 CSA
├── Stage 3 流水线寄存器
│   ├── 50bit 加法器
│   ├── LZD / 规格化器
│   └── 舍入逻辑
└── 输出组装
```
