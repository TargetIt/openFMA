# 阶段 2：支持 FP8×4（四路并行）设计文档

> **增量开发：在阶段 1 基础上新增 FP8×4 FMA 支持，同时完整保留 FP32 和 FP16 FMA 功能。**

## 1. 模块概述

本阶段在阶段 1 基础上增量开发，支持四路并行 FP8（E4M3 格式）FMA 运算：

$$result^{(i)}_{[7:0]} = A^{(i)}_{[7:0]} \times B^{(i)}_{[7:0]} + C^{(i)}_{[7:0]}, \quad i \in \{0,1,2,3\}$$

四路 FP8 操作数打包在 32 位输入中，每路 8 位。

通过 `mode[1:0]` 信号选择精度模式：
- `2'b00`：FP32 FMA（阶段 0 功能）
- `2'b01`：FP16 FMA（阶段 1 功能）
- `2'b10`：FP8×4 FMA（本阶段新增）

## 2. 模块接口

### 输入端口

| 端口名 | 位宽 | 描述 |
|--------|------|------|
| `clk` | 1 | 系统时钟 |
| `rst_n` | 1 | 异步低电平复位 |
| `valid_in` | 1 | 输入数据有效信号 |
| `mode` | [1:0] | 精度模式：00=FP32，01=FP16，10=FP8×4 |
| `A` | [31:0] | 操作数 A（FP8×4 打包：`A[7:0]`, `A[15:8]`, `A[23:16]`, `A[31:24]`） |
| `B` | [31:0] | 操作数 B（FP8×4 打包） |
| `C` | [31:0] | 操作数 C（FP8×4 打包） |

### 输出端口

| 端口名 | 位宽 | 描述 |
|--------|------|------|
| `result` | [31:0] | 运算结果（FP8×4 结果同样打包） |
| `valid_out` | 1 | 输出数据有效信号 |
| `overflow` | 1 | 上溢标志 |
| `underflow` | 1 | 下溢标志 |
| `inexact` | 1 | 非精确结果标志 |

## 3. FP8 位域定义（E4M3 格式）

| 字段 | 位段 | 位宽 | 说明 |
|------|------|------|------|
| 符号位 | `[7]` | 1 bit | 0 = 正，1 = 负 |
| 指数位 | `[6:3]` | 4 bit | bias = 7 |
| 尾数位 | `[2:0]` | 3 bit | 含隐含 1（规格化数） |

## 4. 四路 FP8 数据通路

### 4.1 数据解包

```
A_fp8[0][7:0] = A[7:0]
A_fp8[1][7:0] = A[15:8]
A_fp8[2][7:0] = A[23:16]
A_fp8[3][7:0] = A[31:24]
```

### 4.2 每路乘法

- 尾数乘法：`{1, A[2:0]} × {1, B[2:0]}` → 8bit 乘积
- 复用 FP32 乘法器低 8 位
- 四路乘积独立计算

### 4.3 指数处理

- FP8 指数加法：`exp_sum = A[6:3] + B[6:3] - 7`
- FP8 → FP32 指数扩展：`exp_fp32[7:0] = fp8_exp_sum[3:0] + 8'd113`

## 5. CSA 树复用方案

- 阶段 0 的 3-2 CSA 扩展为 4-2 CSA（增加一级压缩）
- 四路 FP8 乘积对齐后送入共享 CSA 树
- 每路独立使用 CSA + 加法器完成 FMA

## 6. 运算器复用汇总

| 运算器 | FP32 | FP16 | FP8×4 |
|--------|------|------|-------|
| 48bit 乘法器 | ✅ 全用 | ✅ 低22位 | ✅ 低8位×4 |
| 50bit 移位器 | ✅ | ✅ | ✅ |
| 3-2 CSA（50bit） | ✅ | ✅ | ✅→4-2扩展 |
| 50bit 加法器 | ✅ | ✅ | ✅ |
| 规格化/LZD | ✅ | ✅ | ✅ |

## 7. 特殊值处理

| 条件 | 处理 |
|------|------|
| FP8 NaN（E4M3: exp=15, man=7） | 输出 NaN |
| FP8 零（exp=0, man=0） | 输出零 |
| 所有 FP32/FP16 特殊值 | 同阶段 0/1 |

## 8. 模块层次

```
fma_fp8x4_stage2
├── 格式解析前端（mode 分发）
│   ├── FP32 字段提取
│   ├── FP16 字段提取 + FP32 扩展
│   └── FP8×4 解包 + 四路字段提取
├── FP32/FP16 FMA 通路（复用阶段 1）
├── FP8×4 四路独立 FMA 通路
│   ├── 每路：乘法 → 对阶 → CSA → 加法 → 规格化/舍入
│   └── 共享乘法器低位段
└── 输出组装（mode 选择输出格式）
```
