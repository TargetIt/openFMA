# 阶段 3：混合精度累加模式设计文档

> **增量开发：在阶段 2 基础上新增混合精度累加模式，同时完整保留 FP32、FP16 和 FP8×4 FMA 功能。**

## 1. 模块概述

本阶段在阶段 2 基础上增量开发，新增两种混合精度累加模式：

### 模式 A：FP16×FP16 → FP32 累加

$$acc_{[31:0]} = acc_{[31:0]} + \left( A_{[15:0]} \times B_{[15:0]} \right)_{\text{extend to FP32}}$$

### 模式 B：FP8×4 → FP32 累加

$$acc_{[31:0]} = acc_{[31:0]} + \sum_{i=0}^{3} \left( A^{(i)}_{[7:0]} \times B^{(i)}_{[7:0]} \right)_{\text{extend to FP32}}$$

## 3. 累加通路设计

### 3.1 累加寄存器

- `acc_reg[31:0]`：内部累加寄存器
- 复位值 = `acc_in[31:0]`（由 `acc_en` 初始化）
- 写回条件：`acc_en & valid_out`

### 3.2 模式 A 数据流（FP16→FP32 Acc）

```
FP16_product → extend_to_FP32 → FP32_adder(product + acc_reg) → acc_reg
```

- 尾数扩展：`product_mantissa[22:0] = {fp16_product[21:0], 1'b0}`
- 指数修正：`exp_out = A[14:10] + B[14:10] + 8'd97`

### 3.3 模式 B 数据流（FP8×4→FP32 Acc）

```
FP8_product[0:3] → extend_to_FP32 → CSA_tree → FP32_adder(sum + acc_reg) → acc_reg
```

- 每路尾数扩展：`product_mantissa[22:0] = {fp8_product[i][7:0], 15'b0}`
- 指数修正：`exp_out = A[6:3] + B[6:3] + 8'd113`

## 4. 精度扩展对照表

| 源格式 | 尾数位宽 | 扩展后尾数 `[22:0]` | 指数偏置修正 |
|--------|---------|-------------------|------------|
| FP16 | 10bit | `{product[21:0], 1'b0}` | `+112` |
| FP8 (E4M3) | 3bit | `{product[7:0], 15'b0}` | `+120` |

## 5. 溢出保护

- 累加结果超出 FP32 范围时，`overflow` 信号拉高
- `acc_reg` 保持饱和值 `32'h7F800000`（+Inf）

## 6. 运算器复用汇总

| 运算器 | FP32 | FP16 | FP8×4 | Acc 模式 |
|--------|------|------|-------|---------|
| 48bit 乘法器 | ✅ | ✅ | ✅ | ✅ 复用 |
| 50bit 移位器 | ✅ | ✅ | ✅ | ✅ |
| 3-2/4-2 CSA | ✅ | ✅ | ✅ | ✅ |
| 50bit 加法器 | ✅ | ✅ | ✅ | ✅ 累加器 |
| 规格化/LZD | ✅ | ✅ | ✅ | ✅ |

## 7. 模块层次

```
fma_acc_stage3
├── 格式解析前端（mode 分发）
├── FP32/FP16/FP8×4 FMA 通路（复用阶段 2）
├── 累加通路
│   ├── FP16→FP32 精度扩展
│   ├── FP8→FP32 精度扩展 + CSA 树
│   ├── FP32 加法器（product + acc_reg）
│   └── 累加寄存器 acc_reg

```
