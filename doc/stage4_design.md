# 阶段 4：FP16 通路兼容 INT8 设计文档

> **增量开发：在阶段 3 基础上新增 INT8 MAD 支持，同时完整保留 FP32 FMA、FP16 FMA、FP8×4 FMA 和混合精度累加的全部功能。**

## 1. 模块概述

本阶段在阶段 3 基础上增量开发，新增 INT8 乘加（MAD）运算：

**FP16 模式（`mode[0] = 0`）：**

$$result_{[15:0]} = A_{[15:0]} \times B_{[15:0]} + C_{[15:0]}$$

**INT8 模式（`mode[0] = 1`）：**

$$result_{[15:0]} = A\_int_{[7:0]} \times B\_int_{[7:0]} + C\_int_{[15:0]}$$

## 2. 模块接口

### 完整模式编码表

| `mode[2:0]` | 操作模式 |
|-------------|---------|
| `3'b000` | FP32 FMA |
| `3'b001` | FP16 FMA |
| `3'b010` | FP8×4 FMA |
| `3'b100` | FP16×FP16 → FP32 Acc |
| `3'b101` | FP8×4 → FP32 Acc |
| `3'b011` | INT8 MAD（本阶段新增） |

### 新增/修改端口

| 端口名 | 位宽 | 描述 |
|--------|------|------|
| `mode` | [2:0] | 完整 3-bit 模式编码 |
| `A` | [31:0] | INT8 模式使用 `A[7:0]` |
| `B` | [31:0] | INT8 模式使用 `B[7:0]` |
| `C` | [31:0] | INT8 模式使用 `C[15:0]` |
| `result` | [31:0] | INT8 结果在 `[15:0]` |

## 3. INT8 位域定义

| 字段 | A\_int | B\_int | 乘积（有符号扩展） | C\_int |
|------|--------|--------|-----------------|--------|
| 符号位 | `[7]` | `[7]` | `[15]` | `[15]` |
| 数值位 | `[6:0]` | `[6:0]` | `[14:0]` | `[14:0]` |

## 4. INT8 乘法器复用方案

### 4.1 复用 FP16 乘法器

- FP16 乘法器输入位宽为 11bit（含隐含位）
- INT8 模式下：
  - `mul_in_a[10:0] = {A_int[7], A_int[7:0], 2'b0}`（符号扩展到 11bit）
  - `mul_in_b[10:0] = {B_int[7], B_int[7:0], 2'b0}`
- `mode[0]` 控制乘法器内部是否启用补码符号扩展路径

### 4.2 INT8 结果符号扩展与加法

- 乘积符号位：`prod_sign = A_int[7] ^ B_int[7]`
- 扩展：`product[15:0] = {{8{prod_sign}}, raw_product[7:0]}`
- 加法器输入：`product[15:0] + C_int[15:0]`
- 结果截断到 `[15:0]`

## 5. 溢出检测

- INT8 乘积范围：`[-128×128, 127×127] = [-16384, 16129]`
- 加上 C 后若超出 `[-32768, 32767]` 则置 `overflow`

## 6. 运算器复用汇总

| 运算器 | 阶段 0 | 阶段 1 | 阶段 2 | 阶段 3 | 阶段 4 |
|--------|--------|--------|--------|--------|--------|
| 48bit 乘法器 | ✅ 全用 | ✅ 低22位 | ✅ 低8位 | ✅ 复用 | ✅ 低16位（INT8） |
| 50bit 移位器 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 3-2 CSA（50bit） | ✅ | ✅ | ✅→4-2扩展 | ✅ | ✅ |
| 50bit 加法器 | ✅ | ✅ | ✅ | ✅（累加器） | ✅ |
| 规格化/LZD | ✅ | ✅ | ✅ | ✅ | ❌（INT8不需要） |

## 7. 模块层次

```
fma_top_stage4
├── 格式解析前端（mode 分发）
│   ├── FP32 字段提取
│   ├── FP16 字段提取 + FP32 扩展
│   ├── FP8×4 解包 + 四路字段提取
│   └── INT8 有符号数提取（新增）
├── FP32/FP16/FP8×4 FMA 通路（复用阶段 2/3）
├── 累加通路（复用阶段 3）
├── INT8 MAD 通路（新增）
│   ├── 有符号 8x8 乘法器（复用 FP16 乘法器低位）
│   ├── 符号扩展到 16bit
│   └── 16bit 加法器
└── 输出选择（mode 选择输出格式）
```

## 8. 三级流水线对齐

| 流水线阶段 | FP32/FP16/FP8 | INT8 |
|-----------|--------------|------|
| Stage 1 | 乘法 | 有符号 8×8 乘法 |
| Stage 2 | 对阶/CSA | 符号扩展 + 加法 |
| Stage 3 | 规格化/舍入 | 溢出检测 + 输出 |
